# Pasta de Imagens — Grupo "Os Antónios"

Coloque aqui as seguintes imagens para o site funcionar corretamente:

## Imagens obrigatórias

| Ficheiro             | Descrição                                              |
|----------------------|--------------------------------------------------------|
| `hero.jpg`           | Fotografia da Ermida de Santo António (imagem de fundo do hero) |
| `logo.png`           | Logótipo circular do grupo (fundo transparente)       |
| `bandeira.jpg`       | Fotografia da bandeira histórica do grupo             |

## Pasta da galeria

Crie uma sub-pasta chamada `galeria/` e coloque lá as fotografias dos eventos.
Os nomes esperados pelo site são:

| Ficheiro                    | Descrição                    |
|-----------------------------|------------------------------|
| `galeria/aniversario-2024.jpg` | Aniversário 2024          |
| `galeria/piquenique-2024.jpg`  | Piquenique no Rio 2024    |
| `galeria/aniversario-2005.jpg` | 25.º Aniversário 2005     |
| `galeria/procissao.jpg`        | Procissão de Santo António |
| `galeria/piquenique-2023.jpg`  | Piquenique no Rio 2023    |
| `galeria/aniversario-2020.jpg` | 40.º Aniversário 2020     |

Para adicionar mais fotografias, edite o ficheiro `index.html` e duplique um bloco
`<div class="gallery-item">` na secção Galeria, ajustando o caminho da imagem e o
atributo `data-caption`.

## Formatos suportados

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

**Dica:** Para melhor desempenho, optimize as imagens para web antes de as colocar
(recomendado: largura máxima 1200px, qualidade ~80%).
